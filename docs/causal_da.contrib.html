class AugmentingMultiAssessmentEvaluator(x_tr, y_tr, x_te, y_te, metrics, augment_size, namespace='', run_logger=None)¶

Bases: causal_da.api_support.evaluator.AugmenterEvaluatorBase

Evaluate an augmenter by running it once on a data and probing multiple assessment metrics.

Parameters

  • x_tr (ndarray) – target domain training data predictor variables (shape (n_train, n_dim)).

  • y_tr (ndarray) – target domain training data predicted variables (shape (n_train, 1)).

  • x_te (ndarray) – target domain test data predictor variables (shape (n_test, n_dim)).

  • y_te (ndarray) – target domain test data predicted variables (shape (n_test, 1)).

  • metrics (Iterable[AugAssessmentBase]) – the metrics to assess the augmenter.

  • augment_size (Union[float, int, None]) – the desired size of the augmentation used for the assessment.

  • namespace (str) – the namespace of the metric to be prepended.

  • run_logger – the logger to save the results of the assessments.

Parameters

  • augmenter – the augmenter.

  • epoch (Optional[int]) – the epoch when this evaluation is run in the training loop (None if outside a training loop).

class TargetDomainsAverageEvaluator(tar_tr, tar_te, assessments, namespace='', run_logger=None)¶

Bases: causal_da.api_support.evaluator.AugmenterEvaluatorBase

Evaluate the augmenter by performing the assessment on multiple domains and taking the average.

Parameters

  • tar_tr (Tuple[ndarray, ndarray, ndarray]) – tuple of target domain training data containing (x_tr, y_tr, c_tr).

  • tar_te (Tuple[ndarray, ndarray, ndarray]) – tuple of target domain testing data containing (x_te, y_te, c_te).

  • assessments (Iterable[StandardAssessmentBase]) – the assessment objects to assess the augmenter.

  • metrics – the metrics to assess the augmenter.

  • namespace (str) – the namespace of the metric to be prepended.

  • run_logger – the logger to save the results of the assessments.

Note

c_tr and c_te are the indicators of the domain indices (this class averages the performance on multiple domains).
