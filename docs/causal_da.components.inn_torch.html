causal_da.components.inn_torch.glow_nonexponential¶

class GlowNonExponential(depth, dim, n_hidden=20)¶

Bases: torch.nn.modules.module.Module

A Glow model based on the exponential-activation-free affine coupling layers.

Parameters

  • depth (int) – The number of the (invertible linear, affine coupling) flow layer pairs to stack.

  • dim (int) – The dimension of the input data.

  • n_hidden (int) – The number of the hidden units of the parametrization of $s$ and $t$ in each affine coupling layer (the number of layers is fixed at one-hidden-layer).

Note

Since we employ invertible linear layers, we do not require dimension-swap layers in between the affine coupling layers.
